#### 命令行窗口输入 npm install vue-cli -g 出现错误
![](https://github.com/qzssl/sell/blob/master/issue/images/)AUL4R`K03}K%3BHFJ(CAO7.png)
重装node
#### 安装全局vue-cli
```
npm install -g @vue/cli
```
#### 安装webpack
```
npm install webpack -g
```
#### 创建
```
vue create sell
```
#### 启动
```
 npm run serve
```
#### 使用express模拟mock数据
```
npm install express --save
```
