#### 命令行窗口输入 npm install vue-cli -g 出现错误
![](./images/AUL4R%60K03%7DK%253BHFJ(CAO7.png)
>解决：重装node
#### 安装全局vue-cli
```
npm install -g @vue/cli
```
#### 安装webpack
```
npm install webpack -g
```
#### 创建
```
vue create sell
```
#### 启动
```
 npm run serve
```
#### 使用express模拟mock数据
```
npm install express --save
```
